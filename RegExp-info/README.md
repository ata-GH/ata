<p><strong>RegExp学习</strong></p>
<hr/>
<ul>
    <li>
        <strong>RedExp的创建方式</strong><br/>
        1.第一种方式，隐式创建<br/>
        <pre>
        var reg=/pattern/[flag];
        //pattern(必需参数)是按照正则表达式的模式编写的匹配规则，flag (可选参数)匹配模式的修饰符，主要有以后几种标识符 g 全局标志/ i 忽略大小写/ m 用作多行标志
        //例：
        var reg =/^1[8|3|6][\d]{9}$/g;
        </pre>
        2.第二种方式，显示创建<br/>
        <pre>
        var reg=new RegExp('^1[8|3|6][\d]{9}$','gmi');
        </pre>
    </li>
    <li>
        <strong>RedExp的可选参数flag成员</strong><br/>
        常用属性（参数形式）<br/>
        1.global 全局标志（一个正则表达式的只读属性）,当正则对象设置了g修饰符时，此属性为true。<br/>
        2.ignoreCase 忽略大小写（一个正则表达式的只读属性）,当正则对象设置了i修饰符时，此属性为true。<br/>
        3.multiline 用作多行标志（一个正则表达式的只读属性）,当正则对象设置了m修饰符时，此属性为true。<br/>
        不常用属性（不是参数形式）<br/>
        4.lastIndex  该方法用于返回字符串中下一个匹配串的位置（且当global属性为true时有效）。<br/>
        5.source   返回的是正则表达式的模式（不包含修饰符）。<br/>
        <pre>
            var re=/[nd]o+dle/gmi;
            console.log(re.source);  
            //输出 [nd]o+dle
        </pre>
    </li>
    <li>
        <strong>RedExp的pattern语法（匹配模式）</strong><br/>
        patter的元字符大概分为下面几种组合而成<br/>
        1.限定符(用于指定请前面的字符和组合项连续出现多少次)<br/>
        <pre>
        //分为下面几种情况
        {n} 表示出现前面的字符或者组合项目出现了n次
        {n,m}表示出现前面的字符或者组合项目出现了n到m次
        </pre>
        2,选择匹配符<br/>
        <pre>
        +   表示前面的字符或者组合项目出现了一到任意多次
        *   表示前面的字符或者组合项目出现0到任意多次
        ?   表示前面的字符或者组合项目出现0次或者一次
            /a[1-9]?/gi ; //就是指a过后1-9之间的数字只出现一次
        </pre>
        3.分组组合和反向引用符<br/>
        <pre>
        [];分组字符匹配符
        例: [a-z]表示a-z之间的任意一个字符
            [0-9]表示0-9之间的任意一个字符
            [A-Z]表示A-Z之间的任意一个字符
            [abcdf]表示匹配abcdf之间的任意一个字符
        ^ ;反向引用字符串 
        例: [^a-f]表示匹配不是a-f之间的任意一个字符
            [^abcd]表示匹配不是abcd之间的人一个字符
        | and的意思
        例： [a|d|f]表示连续匹配是adf的字符串
            （a|d|f）表示连续匹配是adf的字符串
        </pre>
        4.特殊字符<br/>
        <pre>
        \xxx 	查找以八进制数 xxx 规定的字符。
        \xdd 	查找以十六进制数 dd 规定的字符。
        \uxxxx 	查找以十六进制数 xxxx 规定的 Unicode 字符。
        </pre>
        5.字符匹配符<br/>
        <pre>
        . 	查找单个字符，除了换行和行结束符。
        \w 	查找单词字符。
        \W 	查找非单词字符。
        \d 	查找数字。
        \D 	查找非数字字符。
        \s 	查找空白字符。
        \S 	查找非空白字符。
        \b 	匹配单词边界。
        \B 	匹配非单词边界。
        \0 	查找 NUL 字符。
        \n 	查找换行符。
        \f 	查找换页符。
        \r 	查找回车符。
        \t 	查找制表符。
        \v 	查找垂直制表符。
        
        </pre>
        6.定位符<br/>
        <pre>
        n$ 	匹配任何结尾为 n 的字符串。
        ^n 	匹配任何开头为 n 的字符串。
        ?=n     匹配任何其后紧接指定字符串 n 的字符串。
        ?!n     匹配任何其后没有紧接指定字符串 n 的字符串。
        </pre>
    </li>
    <li>
    <strong>支持正则表达式的string方法</strong><br/>
    <pre>
    search 	检索与正则表达式相匹配的值。 (返回匹配的子串的起始位置若没有就返回-1)并且总是从起始位置开始检索
    match 	找到一个或多个正则表达式的匹配。 	
    replace 	替换与正则表达式匹配的子串。 
    split 	把字符串分割为字符串数组。
    </pre>
    </li>
</ul>
